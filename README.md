# Simple-CLI-Calculator
Hi there! I'm a 17-year-old developer starting my journey in programming. This calculator is my first step toward becoming a Machine Learning Engineer.
🚀 Key Features:
Error Handling: Uses try-except blocks to prevent crashes from invalid inputs (like letters instead of numbers).
Safe Division: Includes specific checks for ZeroDivisionError.
Continuous Loop: The program runs until the user decides to exit.
Supported Operations: Addition, Subtraction, Multiplication, Division, Exponentiation, and Modulo.
🛠 Tech Stack:
Language: Python 3.x
Concepts applied: Loops, Conditional Logic, Data Types (float), Exception Handling.
